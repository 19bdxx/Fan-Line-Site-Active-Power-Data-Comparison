# 模块1：风机-集电线路功率关系分析

本模块分析峡阳B风电场风机 SCADA 功率汇总与集电线路 CT 测点功率之间的关系，重点分析三条集电线路（丙/丁/戊）的传输损耗差异。

## 文件说明

| 文件 | 说明 |
|------|------|
| `提取场站集电线路-全站有功.py` | **步骤1**：从原始 SCADA 数据提取集电线路、全站功率字段，并计算风机有功汇总（S1/S2策略），输出带 `_with_sum` 后缀的 CSV |
| `功率层级关系分析.py` | **步骤2**：主分析脚本，生成时序、散点、差值分布、分功率区间等图表（图01-08）|
| `功率层级关系分析-补充图表.py` | **步骤3**：传输损耗深度分析，生成分线路对比、月度变化、消耗状态分析等图表（图15-24）|
| `分析报告_风机集电线路全站功率层级关系.md` | 完整分析报告 |

## 输出

- 图表：[`../DATA/峡阳B/analysis_output/`](../DATA/峡阳B/analysis_output/)（图01-08, 15-24）
- 文字报告：`../DATA/峡阳B/analysis_output/analysis_report.txt`

## 运行顺序

```bash
python 提取场站集电线路-全站有功.py   # 需要原始 SCADA 源数据
python 功率层级关系分析.py
python 功率层级关系分析-补充图表.py
```
