# 待分析情况



每台风机包含5类数据，以63号风机为例，包括'STATUS_#63', 'ACTIVE_POWER_#63', 'REACTIVE_POWER_#63', 'WINDSPEED_#63', 'WINDDIRECTION_#63'，分别是状态码、有功、无功、风速、风向。

但是从时序数据分析来看，每台风机的这5类数据存在连续时间内保持不变的情况，我不知道这些数据是什么情况，是不是通信链路断了，重复上一个时刻的数据，还是其他的什么情况。我已经将所有风机连续时间相同的重复值组合、开始时间、结束时间、持续长度统计出来了，就是“联合重复值检测结果.CSV”文件。

我前期做了一些分析，初步总结了三类异常，但是不确定总结的准不准确，可以给你提供一个思路。

| **异常类型** | **描述**                                                     | **原因初判**               | **涉及范围** |
| ------------ | ------------------------------------------------------------ | -------------------------- | ------------ |
| 第一类       | **所有**风机在同一时间段内出现数据重复，持续时长较短。       | 通讯链路整体断开           | 全场         |
| 第二类       | **部分**风机在同一时间段内出现数据重复，持续时长较长。       | 通讯链路部分断开           | 多台风机     |
| 第三类       | **单台**风机在连续时间段内数据重复，状态码可能为发电/无通讯/运维等，持续时长最长。 | 运维操作、设备掉线或假数据 | 单台风机     |
